# magronet-poc

maguro（中古車推薦・契約管理システム）の **車両検索画面モックアップ** を起点にした個人PoC。

## 中身

- `index.html` — 中古車検索画面のモックアップ（営業が在庫を絞り込み顧客へ提案するUI）
  - Tailwind CDN + Noto Sans JP / 配色 navy `#002047` + sky `#00A0E9`
  - フィルタ（メーカー/ボディ/走行距離/鑑定点/支払総額/都道府県 等）・比較バー・提案モーダル

## 見る

```bash
open index.html
```
